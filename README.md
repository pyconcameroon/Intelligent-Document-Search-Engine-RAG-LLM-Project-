# Intelligent Doc Search  A portfolio project demonstrating Retrieval-Augmented Generation (RAG) with Python, FastAPI, LangChain, and Docker.  ## Features  - Ingests PDF documents - Splits text into chunks - Creates embeddings using OpenAI or HuggingFace - Stores vectors locally in ChromaDB - Exposes a FastAPI endpoint for question answering
